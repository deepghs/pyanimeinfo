Installation
===================

``pyanimeinfo`` is currently hosted on PyPI. It required python >= 3.7.

You can simply install pyanimeinfo from PyPI with the following command:

.. code:: shell

    pip install pyanimeinfo

You can also install with the newest version through GitHub:

.. code:: shell

    pip install -U git+https://github.com/deepghs/pyanimeinfo.git@main

of ``pyanimeinfo`` should be shown.
